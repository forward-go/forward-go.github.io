дёҖгҖҒй…ҚзҪ®зұ»

дҪҝз”Ё@ConfigurationжіЁи§Јзұ»

1. еҜје…ҘйўқеӨ–зҡ„й…ҚзҪ®зұ»

дёҚйңҖиҰҒжҠҠжүҖжңүзҡ„й…ҚзҪ®ж”ҫе…ҘеҚ•зӢ¬зҡ„дёҖдёӘзұ»дёӯгҖӮ

йўқеӨ–й…ҚзҪ®еҜје…Ҙж–№ејҸ

  • дҪҝз”Ё@ImportжіЁи§ЈеҜје…ҘйўқеӨ–зҡ„й…ҚзҪ®зұ»
  • дҪҝз”Ё@ComponentScanжіЁи§ЈиҮӘеҠЁжү«жҸҸSpring componentsпјҢеҢ…жӢ¬@Configurationзұ»

2. й…ҚзҪ®ж–№жі•

  • XMLй…ҚзҪ®

еҰӮеҝ…йЎ»дҪҝз”ЁXMLй…ҚзҪ®пјҢеҲҷе»әи®®ж–°е»әдёҖдёӘ@Configurationзұ»пјҢз„¶еҗҺдҪҝз”Ё@ImportResourceжіЁи§ЈеҠ иҪҪXMLй…ҚзҪ®ж–Үд»¶

  • еҹәдәҺJavaзҡ„й…ҚзҪ®пјҲжҺЁиҚҗпјү

3. иҮӘеҠЁй…ҚзҪ®

Spring Bootдјҡж №жҚ®дҪ еј•е…Ҙзҡ„jarеҢ…дҫқиө–иҮӘеҠЁй…ҚзҪ®дҪ зҡ„Springеә”з”ЁгҖӮ

еҗҜеҠЁж–№ејҸпјҡж·»еҠ @EnableAutoConfigurationжҲ–иҖ…@SpringBootApplicationжіЁи§ЈеҲ°дҪ зҡ„@Configurationзұ»д№ӢдёҖ

3.1 жӣҝд»ЈиҮӘеҠЁй…ҚзҪ®

еҸҜд»ҘйҖҡиҝҮеҠ е…ҘвҖқвҖ“debugвҖқеҸӮж•°ејҖеҗҜи°ғиҜ•ж—Ҙеҝ—иҫ“еҮәеҲ°жҺ§еҲ¶еҸ°жҹҘзңӢзӣ®еүҚеә”з”Ёзҡ„иҮӘеҠЁй…ҚзҪ®

3.2 зҰҒз”Ёзү№е®ҡзҡ„иҮӘеҠЁй…ҚзҪ®зұ»

  • жҹҘжүҫеҲ°дҪ дёҚйңҖиҰҒеә”з”Ёзҡ„иҮӘеҠЁй…ҚзҪ®зұ»пјҢйҖҡиҝҮй…ҚзҪ®@EnableConfigurationзҡ„вҖқexcludeвҖқеҸӮж•°зҰҒз”Ёе®ғгҖӮ

жіЁж„Ҹпјҡ еҰӮжһңжҢҮе®ҡзҡ„й…ҚзҪ®зұ»дёҚеңЁclasspathи·Ҝеҫ„дёӯпјҢйңҖжҢҮе®ҡзұ»зҡ„е…Ёйҷҗе®ҡи·Ҝеҫ„

  • иҝҳеҸҜд»ҘйҖҡиҝҮspring.autoconfigure.excludeеұһжҖ§жҺ’йҷӨ

дәҢгҖҒSpring Beansе’Ңдҫқиө–жіЁе…Ҙ

еңЁдҪҝз”ЁSpring Bootж—¶еҸҜд»ҘиҮӘз”ұдҪҝз”Ёз¬ҰеҗҲSpringжЎҶжһ¶ж ҮеҮҶзҡ„Beanе’Ңдҫқиө–жіЁе…ҘжҠҖжңҜгҖӮз®ҖеҚ•жқҘиҜҙпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮ@ComponentжіЁи§ЈеҸ‘зҺ°жүҖжңүBeanе’Ңз”Ё@AutowiredжіЁи§ЈеҒҡжһ„йҖ еҮҪж•°жіЁе…Ҙ

дёүгҖҒ@SpringBootApplicationзҡ„дҪҝз”Ё

еҚ•дёӘ@SpringBootApplicationжіЁи§ЈзӣёеҪ“дәҺеҗҜз”ЁдәҶд»ҘдёӢдёүдёӘжіЁи§Јпјҡ

  • @EnableAutoConfiguration
  • @ComponentScan
  • @Configuration
